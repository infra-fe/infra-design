## zh-CN

可以通过设置 `type` 为 `NO_LOG` 选择另一种风格的图片。

## en-US

You can choose another style of `type` by setting image to `NO_LOG`.

```tsx
import { Empty } from 'infrad';

const App: React.FC = () => <Empty type="NO_LOG" />;

export default App;
```
