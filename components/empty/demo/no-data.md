## zh-CN

可以通过设置 `type` 为 `NO_DATA` 选择另一种风格的图片。

## en-US

You can choose another style of `type` by setting image to `NO_DATA`.

```tsx
import { Empty } from 'infrad';

const App: React.FC = () => <Empty type="NO_DATA" />;

export default App;
```
