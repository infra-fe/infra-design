## zh-CN

可以通过设置 `type` 为 `NO_APP` 选择另一种风格的图片。

## en-US

You can choose another style of `type` by setting image to `NO_APP`.

```tsx
import { Empty } from 'infrad';

const App: React.FC = () => <Empty type="NO_APP" />;

export default App;
```
